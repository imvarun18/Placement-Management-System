What is this System?

The Placement Management System is a comprehensive platform designed to manage student information for college placement drives. It helps track student records, manage registrations, and maintain placement statistics.

Features

  • Add and manage student records
  • Track student information
  • Search and filter students
  • View placement statistics
  • Real-time data updates
  • Responsive design

Technology Stack

  • Backend: Spring Boot 3.3.2, MongoDB
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Database: MongoDB
  • Java Version: 17
